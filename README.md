# minima

====


## Description
[Minima!]はCSSミニマムフレームワークです。約6KBの中に現代のサイト制作で欠かせないハンバーガーメニューやレスポンシブ機能など機能を厳選しています。
ユーザー・インターフェースに優れたカラムデザインは、一見の価値のあるグリッドレスポンシブ＋αカラムに複層プルダウンメニューを実装。
又、機能拡張用のCSS（約14KB）は、複数のソーシャルアイコンを1つにまとめてスプライト処理を施し、更にGZIP圧縮を施したSVGZにjavascriptを用いずに機能するトグルボタンなどがあり、importantも使用していませんので全部合わせて約20KB。
50KB制限のあるAMP対策用の雛型ページセットとして利用もできるのでおすすめです。


## Browser Support

IE9+,Chrome[latest],Safari[latest],Firefox[latest]
